# Ultimate-Number-Guessing-Game-Java
Advanced Java console-based number guessing game



## Description
An advanced Java console-based number guessing game.
The game generates a random number based on difficulty levels.
The user guesses the number with smart hints.

## Features
- Difficulty levels (Easy / Medium / Hard)
- Random number generation
- Hint system
- Attempt counter
- Play again option

## Technologies Used
- Java
- Random & Scanner classes

## How to Run
1. Compile:
   javac NumberGuessingGame2.java
2. Run:
   java NumberGuessingGame2
